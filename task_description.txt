Реализовать проект с использованием Django и bootstrap фреймворка, а
также библиотек JQuery и highcharts (http://www.highcharts.com/).

Суть проекта: отобразить в графическом виде некие статистические
данные по некоторым абстрактным показателям во времени.

Структура данных:
  datetime - дата
  value1  - значение некоего показателя N1 за указанную дату (Integer).
  value2  - значение некоего показателя N2 за указанную дату (Integer).

Необходимо:
    1. Реализовать управление данными в стандартной Django админке.
    2. Реализовать отображение этих данных на клиенте в виде графика с
использованием библиотеки    highcharts.
       Данные для отображения принимать отдельным запросом по ajax.
       Так же реализовать возможность выбора периода для отображения
(по полю datetime).
    3. Подготовить скрипт заполнения случайными равномерными тестовыми
данными для периода год (~100к)

Примечания:
- для хранения данных - любая реляционная СУБД
- для верстки страниц использовать фреймворк bootstrap.
- Django последняя стабильная версия

плюсом будет
- приложить файл requirements.txt для быстрого развертывания
- предоставить ссылку на репозиторий с кодом
